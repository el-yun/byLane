# /bylane monitor

## 설명

현재 진행 중인 모든 byLane 에이전트의 상태를 2열 그리드 TUI로 실시간 표시한다.
`.bylane/state/*.json` 파일을 1초마다 폴링하여 갱신.

## 실행

```bash
npm run monitor
```

또는:
```bash
node src/monitor/index.js
```
